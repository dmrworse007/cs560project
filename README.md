# Solar System VR/AR Visualization Project

This project is an interactive 3D Solar System visualization developed using A-Frame, AR.js, WebGL, and WebXR. 
It supports both Virtual Reality (VR) and Augmented Reality (AR) modes and was created as part of my CS560 course project.
